# Pipeline

## API Audit Flow

1. Phase 1 — Diagnosis  
   - quick analysis of the process  
   - risks and bottlenecks detection  

2. Phase 2 — Final report generation  
   - structured audit output  
   - automation opportunities  
   - ROI hypotheses  

3. Phase 3 — Reviewer pass  
   - normalize headings  
   - remove artifacts  
   - enforce unified style  
   - retry if output is broken

## Validation

Pipeline includes:

- report normalization
- broken-output detection
- automatic retry
